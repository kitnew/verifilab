# Demo script

1. Seed the database and sign in as `admin` with the configured bootstrap password.
2. On Dashboard, use the onboarding card to explain the four stages.
3. Open **STEM Reasoning Benchmark** and inspect its verifier types and task statuses.
4. Open **Capital of Romania**, then **Evaluate rollouts**; add `Bucharest` and `Cluj`, create the batch, and run it.
5. Open a draft task and move it through **Start work**, **Submit for review**, and **Approve**.
6. Open **Validated STEM Core** under Datasets; show its quality warning and existing `1.0.0` release.
7. Open the release and export **Full release** JSONL.
8. Finish on Jobs and Activity to show operational and audit history.
