# Product workflow

1. Create a project, then author, import, or deterministically generate tasks.
2. Configure an exact-match, numeric, regular-expression, or JSON Schema verifier.
3. Import candidate rollout responses and run an evaluation batch against its verifier snapshot.
4. Move the task through work, submission, review, and approval; comments and transitions enter the audit trail.
5. Add approved tasks to a dataset and inspect quality findings.
6. Create an immutable semantic-versioned release with deterministic train/validation/test splits.
7. Export the full release or one split as JSONL.

Task, evaluation, review, dataset, release, job, and audit views remain linked in the dashboard.
